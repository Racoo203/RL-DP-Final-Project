# RL-DP-Final-Project

## Overview
This repository contains the final project for the MA338 Dynamic Reinforcement Learning course. It implements dynamic programming algorithms for reinforcement learning problems. You can find the python packages necessary in requisites.yml.

## Usage
Run the main script to execute the algorithms: run `FINAL_PROJECT_2026.ipynb`. You can find the report in report/MA338_DynamicRL_Final.pdf.
